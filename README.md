# SeisCore

Real-time seismic monitoring dashboard. Live demo: https://seiscore-frontend.vercel.app

## What it does

Visualizes earthquake data from a .NET backend on an interactive world map. Events are color-coded by magnitude severity. The analytics page provides deeper insight through charts — events over time, severity distribution, most active regions and magnitude vs depth.

## Tech stack

Next.js · TypeScript · Tailwind CSS · Leaflet · ECharts
